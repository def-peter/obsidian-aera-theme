# 代码也可以成为阅读的一部分

正文中的 `行内代码` 应该清楚，但不需要打断句子的节奏。完整代码块则需要更明确的边界，因此 Aera 使用 Monokai 暗色配色，让代码从中文正文中自然独立出来。

```javascript
const aera = {
  强调色: "#1677FF",
  排版: "清晰而舒展",
  内容层级: ["正文", "引用", "提示", "代码"],
  保持专注: true,
};

function 开始写作(主题) {
  return `${主题}，从一个清晰的问题开始。`;
}
```

## 信息结构

| 内容 | 视觉表达 | 阅读作用 |
| --- | --- | --- |
| 正文 | 舒展、克制 | 保持连续阅读 |
| 引用 | 灰色纸张折角 | 区分摘录与观点 |
| Callout | 语义颜色与图标 | 快速识别信息类型 |
| 代码 | Monokai 暗色主题 | 建立清晰边界 |
